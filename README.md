📊 Chennai Business Data Analytics Project
📌 Project Overview

This project was developed during my internship at Adz4Needz. The objective was to collect, clean, analyze, and visualize business data from Chennai across six major industry segments to generate actionable business insights.

The project follows a complete end-to-end data analytics workflow — from web scraping to dashboard reporting.

🏢 Industry Segments Covered

Food

Healthcare

Retail

Education

Services

Local Businesses

All data collected is specific to Chennai city.

🔄 Project Workflow
1️⃣ Data Extraction

Collected raw business data using:

Instant Data Scraper (Chrome Extension)

Selenium WebDriver (Python Automation)

Extracted fields such as:

Business Name

Category

Rating

Review Count

Contact Information

Address

Website Availability

2️⃣ Data Cleaning & Transformation

Tools Used:

Excel

Python (pandas, NumPy)

Tasks Performed:

Removed duplicates and null values

Standardized column names and formats

Cleaned rating & review count formats

Structured datasets for analysis

Combined multiple segment datasets into a unified master dataset

3️⃣ Data Mining & Analytics

Performed in Google Colab (Python)

Exploratory Data Analysis (EDA)

Rating distribution analysis

Review volume trend analysis

Website presence comparison

Business segmentation into 3 performance tiers:

High Performing

Medium Performing

Low Performing

Segmentation was based on rating + review engagement metrics.

4️⃣ Dashboard Development

Tool Used:

Power BI

Created interactive dashboards including:

📌 Total Businesses

⭐ Average Ratings

📝 Total Reviews

🌐 Website Presence Analysis

📊 Segment-wise Performance Comparison

📈 Review Distribution Visualization

The dashboard enables stakeholders to quickly identify:

High-performing categories

Underserved segments

Digital presence gaps
